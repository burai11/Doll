このフォルダについて（素材追加用 / Asset drop-zone）
====================================================

このフォルダは「AI に新しい衣装素材を渡すための一時置き場」です。
アプリ本体（index.html / js / characters）はこのフォルダを一切参照しません。
そのため、このフォルダは空でも、削除しても、アプリの動作には影響しません。

■ 使い方（新しい衣装を追加したいとき）
  1. 追加したい素材画像をこのフォルダに入れる
       - 背景は「緑（グリーンスクリーン）」推奨。AI が自動で透過処理します。
       - 白背景でも可（その場合は removeWhiteBackground で処理）。
       - 画像サイズは他素材と同じキャンバス（784x1312 目安）で、
         キャラクターの立ち位置に合わせて配置してください。
       - 種類ごとにサブフォルダで整理してもOK（例: 婦警/ ごす/ など）。
  2. AI に「素材追加フォルダの◯◯を追加して」と伝える。
  3. AI が透過PNG化して characters/<キャラ>/images/ に保存し、
     items.json / character.json に登録します。

■ 処理後
  取り込みが終わった素材は、このフォルダから削除して空にして構いません。
  最終的な透過PNGは characters/ 側に保存されているため、原本は不要です。
  （バックアップとして残しておいても問題ありません。）

■ 対応カテゴリの例
  帽子(hat) / 服(tops) / スカート(skirt) / ワンピース(dress) /
  アウター(outer) / ブラ(bra) / パンツ(pants) / ソックス(socks) / 腕(armOverlay)
  新しいカテゴリが必要な場合も AI が character.json に追加します。
